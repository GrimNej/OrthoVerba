# Privacy

Unsaved scripts and recognition state exist only in application memory. Reloading or closing the tab removes them from OrthoVerba-owned state. The app uses no application cookies, analytics, remote fonts, backend, localStorage, or sessionStorage. IndexedDB is opened only after the user chooses **Save on this device**. The service worker caches application shell files only.

Browser-selected speech recognition may be processed by a browser-operated remote service. OrthoVerba does not receive, record, or upload microphone audio itself.
